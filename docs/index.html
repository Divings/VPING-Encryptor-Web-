VPING Encryptor

PNG形式で暗号化データを保存する、耐障害型暗号化ツール

概要

VPING Encryptor は、ファイルを AES-GCM で暗号化し、 PNG形式の暗号化コンテナとして保存するソフトウェアです。

Version 4.0.0 では内部フォーマットを刷新し、 ECC（誤り訂正）、SHA-256検証、破損復旧処理を強化しました。

主な機能

  • AES-GCM によるファイル暗号化
  • PNG形式での暗号化データ保存
  • ECC（誤り訂正）による破損耐性向上
  • SHA-256 による整合性検証
  • ブロックチェーン風の操作履歴管理
  • GUIによる暗号化・復号操作

Version 4.0.0

  • PNG暗号化コンテナ内部構造を v3 に刷新
  • ECC適用位置を見直し、PNG内部データ破損への耐性を向上
  • 検証処理と復旧処理を改善
  • 復号時の _decrypted 付与設定を追加
  • 上書き確認ダイアログの ON/OFF 設定を追加
  • 元ファイル削除設定を設定ファイル側で管理

重要: 旧形式との互換性について

VPING Encryptor 4.0.0 では、内部フォーマット v3 への移行に伴い、 旧バージョンで生成された一部PNG暗号化データとの互換性がありません。

旧データを利用する場合は、必要に応じて旧バージョンで復号してください。

内部構造

Version 4.0.0 では、以下のような多層構造でデータを保護します。

元データ
 ↓
AES-GCM 暗号化
 ↓
LZMA 圧縮
 ↓
ECC 付与
 ↓
PNG コンテナ化
